# ts-node-boilerplate

## Description

Typescript node.js express app boilerplate:

- typescript
- node.js
- express
- inversify
- docker
- eslint
- jest
- e2e test setup

## Development

Install: `npm install`

Run app: `npm run start`

E2E tests `npm run test:e2e:local`

Unit tests tests `npm run test:coverage`

Linter tests `npm run lint`
